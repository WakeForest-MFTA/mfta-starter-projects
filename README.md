# MFTA Starter Projects

This repository contains starter projects and reference implementations for the Wake Forest Master of Financial Technology & Analytics (MFTA) program.

## Focus Areas

* Machine learning in finance (credit risk, fraud, forecasting)
* API-driven fintech systems
* AI and agentic workflows
* Data analytics and visualization

## Purpose

These projects are designed to:

* Provide a foundation for student work
* Demonstrate applied fintech concepts
* Serve as a baseline for capstone and independent projects

## Tech Stack

* Python (pandas, scikit-learn)
* Node.js / Express
* PostgreSQL
* React (optional)
* LLM integrations

## AI Usage

Students are expected to use AI tools responsibly and document:

* Prompting approach
* Model usage
* Limitations and validation

## Next Steps

Students should build upon these examples to develop their own projects.
